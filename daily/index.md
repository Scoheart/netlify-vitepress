## 如何使用 Todo List

在每一天的笔记中，你可以使用交互式 TodoList 组件来管理你的任务：

```vue
<TodoList title="今日任务" />
```

组件功能：
- ✅ 添加、完成、删除任务
- 🎯 设置任务优先级（高/中/低）
- 🔍 按状态筛选（全部/进行中/已完成）
- 💾 自动保存到浏览器本地存储
- 📊 任务进度统计
